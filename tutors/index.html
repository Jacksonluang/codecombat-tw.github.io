關於家教

CodeCombat 課程雖然適合孩子們透過遊戲式方式學習程式，不過若是過程中沒有接受任何程式邏輯指導，恐怕會流於遊戲玩樂，而沒有學習到程式的相關概念及內涵

假如家長本身具備程式基礎及經驗，而且有時間及方法，能夠親自引領自己的孩子學習，這是最好，同時也是我們最鼓勵的方式。但是大多數家長並沒有具備程式基礎或概念，即使有，仍可能因為本身時間安排問題或是欠缺教學經驗，而無法給予孩子適時適切的指導

因此，我們希望全國有志於程式教育的同好，加入成為我們的家教師資。家教師資可以選擇使用網路遠端連線教學、到府教學或自行安排上課地點，來教導需要學習程式的孩子

認證規定

先備條件

我們誠摯歡迎有心投入程式教育的同好加入成為師資，加入家教師資不須支付任何加盟費用或事先購買帳號經銷權。我們會引導師資申請開設及管理 CodeCombat 線上教室，家教師資在教學和管理學生方面，是可以完全獨立的。此外，我們會在提供認證、指導及培訓服務時，針對想成為家教師資的夥伴收取服務費用，同時是為確保師資的教學品質

我們的家教課程，現階段使用 Python 程式語言為主。因此，先備條件以熟悉 Python 語言且具備程式設計教學經驗，能立即通過認證者為佳；至於已熟悉 Python 語言，但不具備教學經驗者亦歡迎，不過必須願意接受師資指導及認證；若不熟悉 Python 語言，但有高度興趣且願意嘗試投入程式教育者，不僅同樣需要願意接受師資指導及認證，還必須事先完成我們制訂的 Python 初級師資培訓課程


實施方式

家教師資認證、指導或培訓的進行方式，一律使用 TeamViewer 遠端連線軟體的會議功能，召開線上會議。連線會議前，必須準備筆電加耳機，或是電腦加耳機及麥克風。每回合線上會議，為確保品質與兼顧效益，人數以 3~4 位為限，每回會議時間以 2 小時為限
